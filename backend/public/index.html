Documentación API de Platos Típicos Argentinos

  • Platos
  • Ingredientes

Platos

Obtener todos los platos

GET /api/platos

Obtener plato por nombre

GET /api/platos/:nombre

Crear nuevo plato

POST /api/platos

Actualizar plato

PUT /api/platos/:id

Eliminar plato

DELETE /api/platos/:id

Filtrar platos por tipo

GET /api/platos/tipo/:tipo

Opciones: dulce / salado

Filtrar platos por dificultad

GET /api/platos/dificultad/:dificultad

Opciones: fácil / media / difícil

Filtrar platos por tiempo de cocción

GET /api/platos/tiempo/:tiempo

Escribí el tiempo máximo de la receta en mins.

Ingredientes

Obtener todos los ingredientes

GET /api/ingredientes

Obtener ingrediente por ID

GET /api/ingredientes/:id

Crear nuevo ingrediente

POST /api/ingredientes

Actualizar ingrediente

PUT /api/ingredientes/:id

Filtrar ingredientes por tipo

GET /api/ingredientes/tipo/:tipo

Opciones: vegetal / carne / lácteo / cereal / condimento / otro
